# Changelog

All notable changes to Agentic BDD Flowctl are documented here.

## 0.2.0 - 2026-07-16

First technical-preview release.

### Added

- Source-grounded React/Spring evidence, behavior, witness, and variant pipeline.
- Bounded AI semantic/rule packets with validation and named human approval.
- Application-data planning and confirmation gates for UAT-specific values.
- Traceable holistic Gherkin, Playwright-BDD step delegates, and runtime grounding contracts.
- State-aware agent directives, machine progress events, run history, repair packets, and bundled skill installer.

### Hardened

- Method-aware Spring Security matcher extraction and explicit failure-response handling.
- Nested/index React route composition, Axios instance base URLs, and endpoint query normalization.
- Compiler-owned operation identity and source-approved predicate literals.
- No-shell runner cleanup and suppression of child output from CLI responses.
- Node.js 22/24 CI, clean-package smoke testing, zero production audit findings, SBOM, checksums, and immutable release assets.

### Scope

- This release generates source-grounded happy paths and BDD. Contradiction-based implementation bug finding remains future work.
